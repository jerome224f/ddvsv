# Vyes CRM

A premium, modern CRM designed to streamline lead tracking, contact management, task organization, and more.

## Setup Instructions

1. Install dependencies:
   `npm install`
2. Configure your Supabase environment variables in `.env`
3. Run the development server:
   `npm run dev`
